## 📝 添加学员的方法

### 步骤1：打开配置文件

用记事本打开 `config.js`

### 步骤2：添加学员信息

找到 `STUDENTS_CONFIG` 部分，按格式添加：

```javascript
const STUDENTS_CONFIG = [
    // 已有的学员...
    
    // 添加新学员（复制这个格式）
    {
        studentId: '20240004',        // 学号
        name: '新学员',                // 姓名
        teacherName: '张老师',         // 老师名称
        botId: '7585150421019852842'  // Bot ID
    },
];
```

### 步骤3：保存文件

保存后，新学员就可以用学号登录了。

## 🌐 部署到网上（让学员能访问）

### 方案1：用Vercel（推荐，免费）

**第1步：准备文件**
1. 把 `teacher-bot` 文件夹压缩成zip
2. 或者上传到GitHub

**第2步：部署到Vercel**
1. 访问：https://vercel.com
2. 用GitHub账号登录
3. 点击"New Project"
4. 选择你的GitHub仓库（或上传文件）
5. 点击"Deploy"
6. 等待1分钟，会得到一个网址

**第3步：分享给学员**
把网址发给学员，比如：
```
https://你的项目名.vercel.app
```

学员打开网址 → 输入学号 → 就能用了！

## 🔧 常见问题

### Q1: 学员输入学号后提示"学号不存在"
**解决：**
- 检查 `config.js` 中是否添加了该学号
- 检查学号是否输入正确（区分大小写）

### Q2: 发送消息后没有回复
**解决：**
- 检查扣子Token是否过期（1个月后需要续期）
- 检查Bot ID是否正确
- 打开浏览器F12查看错误信息

### Q3: Token过期了怎么办？
**解决：**
1. 回到扣子 → API管理 → 授权
2. 创建新的个人访问令牌
3. 复制新Token
4. 替换 `config.js` 中的 `token` 值
5. 重新部署

### Q4: 如何查看学员使用情况？
**目前版本：**
- 暂时无法统计（简化版）

**升级版方案：**
- 需要添加后端服务器
- 记录每次对话
- 生成使用报表
